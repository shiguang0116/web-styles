# web-styles

> 一个基于桌面端编写的通用样式文件

#### npm下载

```
npm i web-styles -S
```

#### github地址

> [https://github.com/shiguang0116/web-styles](https://github.com/shiguang0116/web-styles)

#### 文件说明

```
web-styles/
├── dist/
│   ├── common.css      通用样式
│   └── common.min.css  通用样式（压缩后）
├── src/
│   ├── custom.less     变量
│   ├── reset.less      重置样式
│   ├── layout.less     布局样式
│   ├── common.less     定义通用的样式
│   └── index.less      引入样式文件
└── index.html      示例模板
```
